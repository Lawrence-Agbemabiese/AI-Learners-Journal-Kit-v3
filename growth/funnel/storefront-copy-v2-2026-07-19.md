## Long description

You can finish a coding course and still struggle to explain what you actually learned.

A certificate says you attended. It cannot show an employer what you understood, built, fixed, and can explain. The AI Learner's Journal Kit closes that gap with a weekly evidence system: what you studied, what you built, what went wrong, and how you used AI — captured while it's fresh.

Each week's entries become short portfolio stories, and twelve weeks later you have named artifacts — a project case study, a GitHub README, a prompt log, a documented learning record — instead of a blur.

Designed for students in Ghana and across Africa, this pack works especially well for learners in national coding, AI, data, cybersecurity, and digital-skills programs.

## How the app works (product truth)

- **Local and yours:** entries are plain Markdown files saved on your own computer, with built-in search. No account. No cloud. If you stop using the Kit, you keep every entry.
- **Browser interface:** the START HERE launcher opens a friendly page in your web browser; Terminal commands are optional/advanced.
- **Requirements:** a computer with Python 3.9 or newer (free from python.org). The PDFs work on any phone or on paper.
- **AI is optional, bring your own key:** the journal and search work fully offline. If you want AI answers, you connect your own AI provider API key (free-tier options exist); AI providers may require their own signup and usage credits. AI answers are never guaranteed to be accurate.
- **For Claude Code users (optional):** `ai-journal import` turns a local Claude Code session into a draft journal entry, offline — your real prompts pre-filled, the reflection left to you. Claude Code only; no other tools supported.
